### Features

- Data processing topology

- Multi-engine

- Heterogeneous miscalculation

- SQL-like DSL

- Light-weight and pluggable

- Easy to use and expand


### Concepts

- Vertex：data source

- Edge：data pipe

- Env：runtime environment

- Cell：data processing unit
